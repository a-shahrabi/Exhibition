Time to Build Your Own AI!

We'll use Teachable Machine - a free tool from Google

Instructions:

  1. Go to teachablemachine.withgoogle.com
  2. Click "Get Started"
  3. Choose your project type:
    • Image Project (recommended for beginners)
    • Audio Project
    • Pose Project
  4. Create 2 classes for your model. Ideas:
    • Peace sign vs Thumbs up
    • Happy face vs Sad face
    • Clap vs Snap (for audio)
    • Or create your own idea!
  5. Collect at least 20-30 examples for each class
  6. Click "Train Model" and wait
  7. Test your model with the preview
  8. Record a 20-30 second demo showing it working
